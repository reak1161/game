# 超次元バトル - プロダクト概要

## 1. プロダクト概要
超次元バトルは、プレイヤーが役職を選択し、カードを場に配置して連鎖・追加発動・設置効果・エンチャントを組み合わせながら高スコアを狙う、ラウンド制のWebカードゲームである。

本作の「ダメージ」は相手HPを削る概念ではなく、各プレイヤーが生み出す **個人スコア** として扱う。  
同一ルーム内のプレイヤーは、同じ試合文脈の中で自分の発動を進め、その結果を **順番に全員で観戦・比較** する。

## 2. コンセプト
- カード順と属性連鎖で爆発的なスコアを作る
- 発動テンポが加速していく気持ちよさを演出する
- 盤面構築と観戦体験の両方を楽しめる
- プレイヤー同士は直接殴り合わず、各自の最適化結果を比較する

## 3. ゲームの特徴
- 役職選択制
- 攻撃 / 魔法 / 能力カード
- 基礎ステータス + 一時ステータス
- 発動 / 設置 / 消費 / エンチャント
- 属性（炎、水、氷、風、雷、地、闇、無属性）
- 左から順の解決
- 無制限の場
- ルーム共有型の順番観戦
- 演出重視（パーティクル、音、加速再生）

## 4. 目標体験
### プレイ時
- カード配置だけでなく「どの順で発動するか」が重要
- 連続や追加発動でコンボが大きく伸びる
- 毎ラウンド終了時にスコアの伸びを実感できる

### 観戦時
- 全員が同じ再生状態を見られる
- 誰のどのカードがいま発動したか分かりやすい
- 後半ほどテンポが加速し、爽快感が高まる

## 5. 想定モード
### 共有観戦モード（主モード）
- ルームを作成して参加する
- 各プレイヤーはカード配置などの入力を行う
- ラウンドごと、あるいは試合ごとに結果イベント列を生成する
- そのイベント列をルーム内全員で同じ順番・同じ位置から観戦する

### ローカル検証モード（開発・テスト用）
- 1人で試し打ちできる
- カード挙動確認やデバッグ用途を主目的とする
- 製品上は隠し機能または開発モード扱いでもよい

## 6. 初期スコープ
- 2〜4人ルーム
- 5ラウンド制
- 役職3種程度
- 無属性 / 炎 / 水 / 氷 / 風 / 雷の一部カード実装
- 共有観戦の基礎実装
- ログ再生と演出再生

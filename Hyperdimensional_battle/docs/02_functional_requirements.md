# 超次元バトル - 機能要件

## 1. 必須機能一覧
- ルーム管理
- ゲーム進行
- 観戦・再生
- 演出
- データ管理

## 2. 画面要件
- タイトル画面
- ルーム画面
- ゲーム画面
- 結果画面

## 2.5 スクロール要件
- 場のカード表示領域は **横スクロール対応必須** とする
- 手札表示領域は **基本横並び** とし、画面幅不足時や枚数増加時は **横スクロール対応** とする
- ログ表示領域は **縦スクロール対応必須** とする
- 場および手札では、発動順や操作性維持のため **折り返し表示を禁止** する
- 詳細情報パネルは必要に応じて内部スクロールを許可する

## 3. 非機能要件
- 主要ブラウザで動作すること
- 一画面で視認性を保つこと

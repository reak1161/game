  - 「攻守の構え」（攻撃）: 次のうちどれか選ぶ（通常4 / 防御カードとして設置）。

  - 「ライブラリバースト」（呪文）: 手札全捨て→自分以外へ通常〈捨てた枚数×1.5〉ダメージ。
  - 【血の剣】（装備）: ロール攻撃時、相手に出血2を付与。
  - 【血の鎧】（装備）: 追加Def+5。ターン開始時に出血1を付与。
  - 【ふしぎなまもり】（装備）: 特殊ダメージ（炎上/出血など）を受けない。
